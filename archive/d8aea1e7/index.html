Vuido开发

2018-11-06
0

Vuido, a framework for creating native desktop applications using Vue.js.

官网：https://vuido.mimec.org/

开发环境：DeepinLinux

先安装系统依赖

sudo apt install build-essential libgtk-3-dev

全局安装

npm install --global vue-cli

创建项目

vue init mimecorg/vuido-webpack-template my-project

安装项目依赖

cd my-project
npm install

编译运行看效果

npm run build
npm start
